Internship Helper

Notes

The Internship Helper is a full-stack feature for the CSXL site that simplifies how students discover internships and how admins manage listings. It centralizes internship opportunities in one platform, making them easy to browse, update, and curate.

Built for the CSXL community, the system automatically scrapes internship data weekly, stores it in a structured PostgreSQL database, and provides secure tools for admins to review, edit, or remove listings through an intuitive Angular dashboard.

Key Features

  • Automated weekly internship data updates
  • Secure FastAPI endpoints for filtering and management
  • Admin dashboard for adding and editing positions
  • Role-based permissions for data security
  • Responsive, modern UI with Angular and TypeScript

Tech Overview

I led the end-to-end development, integrating scraping logic, database design, API development, and frontend UI into one seamless system.

Tech Stack

  • PostgreSQL
